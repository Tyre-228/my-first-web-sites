Что такое FlexBox?

Flexbox - это целый модуль,содержащий в себе набор свойства и их значений как для родителя flex-контейнера так и для его дочерних элементов flex-элементов.

Основным преимуществом гибкой FLEX разметки являеться возможность изменять шырину,высоту,пространство вокруг и порядок flex-элементов для того чтобы наилучшым образом заполнить доступную пространству flex-контейнера.
